## 🎨 快捷键系统

### 主题切换快捷键
- `Ctrl + 1` - 生活模式
- `Ctrl + 2` - 极客模式
- `Ctrl + 3` - 狂暴模式
- `Ctrl + 4` - Emo模式

### 工具跳转快捷键

### 生活模式动画
- 柔和的色相旋转效果
- 轻微的缩放动画

### 极客模式动画
- 强烈的对比度和亮度变化
- 垂直移动效果
- 1.5秒动画时长

### 狂暴模式动画
- 激烈的旋转和缩放动画

### Emo模式动画
- 情感化的色相和饱和度变化
- 垂直移动配合色相旋转
- 2.0秒动画时长

## 🚀 使用方法

1. **查看快捷键**：点击页面左上角的问号按钮
2. **切换主题**：使用 `Ctrl + 数字键`
3. **跳转工具**：使用 `Alt + 字母键`
4. **关闭帮助**：按 `ESC` 键或点击遮罩层
