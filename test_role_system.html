角色系统测试

测试普通用户和管理员的不同权限

普通用户

user
  • 提交建议和反馈
  • 查看自己的建议
  • 使用所有工具功能
  • 无法查看所有建议
  • 无法管理用户反馈

管理员

admin
  • 所有普通用户权限
  • 查看所有建议和反馈
  • 回复用户建议
  • 更新建议状态
  • 管理用户反馈
